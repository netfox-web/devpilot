AI Provider Secrets

Read-only runtime environment inspection for OpenAI, Gemini, and Claude provider keys.
AI Providers External API Keys
Provider keys are different from DevPilot External API Keys.
AI Provider Keys let DevPilot call OpenAI, Gemini, or Claude from its own runtime. DevPilot External API Keys authenticate external projects calling /api/external/*. Do not give OpenAI, Gemini, or Claude keys to external projects, and do not store provider keys in Git, docs, logs, or frontend bundles. This page only inspects runtime env and does not store secrets, run live pings, or call providers.

What This Page Does
  • Checks whether known provider key env vars are present.
  • Shows only the env var name and a masked preview.
  • Never shows raw provider keys, Authorization headers, or key hashes.
  • Does not save provider keys or write database records.
What This Page Does Not Do
  • No live ping button.
  • No provider key create/update form.
  • No provider calls or worker execution.
  • No provider secret manager or migration in this slice.
